# NeoPot Plugin Template

This is a JavaScript template for creating NeoPot plugins.

## Files

- `info.json` - plugin manifest
- `main.js` - plugin entry point

## Install

Copy this directory, update `info.json`, implement `main.js`, then install the directory from NeoPot's plugin page or package it as a zip.

Installed plugins must include `info.json` and `main.js`.
